# miweb2

Una página web rudimentaria básica.

## Contenido

Este proyecto contiene una página web simple con:

- **index.html**: Estructura HTML de la página
- **styles.css**: Estilos CSS para el diseño
- **script.js**: JavaScript para interactividad básica

## Características

- Diseño responsivo
- Navegación con enlaces internos
- Botón interactivo con contador de clics
- Formulario de contacto básico
- Estilos modernos y limpios

## Cómo ver la página

Para ver la página web, simplemente abre el archivo `index.html` en tu navegador web:

1. Descarga o clona este repositorio
2. Navega a la carpeta del proyecto
3. Abre el archivo `index.html` con tu navegador preferido (Chrome, Firefox, Safari, Edge, etc.)

También puedes usar un servidor web local si lo prefieres:

```bash
# Usando Python 3
python -m http.server 8000

# Usando Node.js (con http-server instalado)
npx http-server

# Usando PHP
php -S localhost:8000
```

Luego visita `http://localhost:8000` en tu navegador.
